Mistral Le Chat, one click away.

Stop switching tabs to talk to Le Chat. This tiny Chrome extension tucks it into the native side panel, so it's right next to whatever you're reading.

Install from GitHub How it works

Install it in four steps.

Install, unpacked
  1. 01
    Download the zip from GitHub.
    Green Code button, then Download ZIP. Or clone, if that's you.
  2. 02
    Double-click to unzip. Keep the folder somewhere stable.
    Finder unzips in place. Don't move the folder afterwards; Chrome remembers the path.
  3. 03
    Open chrome://extensions, flip on Developer mode.
    Top-right toggle. Lets Chrome load extensions outside the store.
  4. 04
    Click Load unpacked, pick the folder.
    Pin the cat to the toolbar. Click it, and Le Chat slides into the right side panel.
